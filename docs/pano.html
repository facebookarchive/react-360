Pano #

A Sphere of 1000m with center located at the local transform origin by default this is position:'absolute'

Panos are images that allows you to view the scene from every angle: above, below, behind and next to you. These are a core image format for VR applications You can create 360 photos by using special 360 camera hardware and these will be usually in the form of Equirectangular images covering the full 360x180 angles an example is

Cubemap images are also supported and to enable this the source url is specified as an array of 6 individual images presented in the order [+x, -x, +y, -y, +z, -z]
